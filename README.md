# Readme

In order to get the most out of a microcontroller, you often need to really dive into the datasheets, 
make your Makefile, and start shifting bits and writing registers in C.

This repo will provide some guides and hello-world samples for a few different architectures.


## Installation necessary tools

first up, make sure to grab these tools:

`sudo apt-get install gcc-avr binutils-avr gdb-avr avr-libc avrdude`
